#1 Lista di comandi base per la gestione dei documenti in Elasticsearch

Creare indice

PUT /nome_indice


Eliminare indici

DELETE /nome_indice


Aggiungere Documento

POST /nome_indice/_doc
{
"key":"value"
}


Richiamare un documento specifico

GET /nome_indice/_doc/id_del_documento


Modifica dei campi di un documento

POST /nome_indice/_update/id_del_documento
{
"doc": { "campo da modificare": "valore" }
}
Se il campo non esiste allora Elasticsearch lo aggiungerà automaticamente


Modifica totale di un documento

POST /nome_indice/_doc/id_del_documento
{
"key":"value"
}
Se il campo non esiste allora Elasticsearch lo aggiungerà automaticamente


Eliminare documenti

DELETE /nome_indice/_doc/id_del_documento
